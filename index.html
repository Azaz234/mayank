Get started with Elfsight Testimonials Slider 1.6.1

Installation

1. Install Elfsight Testimonials Slider WordPress Plugin elfsight-testimonials-slider-cc.zip to your website. Don't know how? Please learn here.

2. Go to Testimonials Slider page in WordPress admin menu.

3. Create a new widget, adjust all settings and get a ready-made shortcode.

Use Testimonials Slider as a WordPress Shortcode



1. Create a new widget

You can use Elfsight Testimonials Slider as a WordPress shortcode. To do this, you have to create a new widget, set all options in the Elfsight editor and save it.

2. Copy-paste the shortcode

You will see the new widget's shortcode in the widgets list. You can copy-paste it into the required place of your website.

Example of the Elfsight Testimonials Slider shortcode:

[elfsight_testimonials_slider id="1"]

Use Testimonials Slider as a
Gutenberg Block

You can use Elfsight Testimonials Slider as a block for Gutenberg editor. Just add it to your page during page building and select one of the available Testimonials Slider widgets from the list.

Use Testimonials Slider as a
WordPress Widget

To use Elfsight Testimonials Slider as a WordPress widget, go to Appearance -> Widgets, and drag Elfsight Testimonials Slider WordPress widget to any widget section. Next, select one of available Testimonials Slider widgets from the list.

Use Testimonials Slider as a
Visual Composer Element

You can use Elfsight Testimonials Slider as a Visual Composer element. Just add it to your page during page building and select one of the available Testimonials Slider widgets from the list.

Options

Option Default Value Description
Testimonial Template Tiled Classic Select one of the 6 predefined templates for the testimonials in your slider. Tiled templates allow you to display several testimonials at ones, depending on the size of your slider. Single templates display individual testimonials. Perfect choice for cases when you need to focus on each individual story.
Layout Slider Choose Slider layout to display testimonials in a feed as part of your page's content, or Grid layout to display them on a separate page or individually.
Testimonials - Here you can add, edit and order your testimonials.
Testimonials: Text - Paste testimonial text to this field
Testimonials: Author Name - Enter the name of the testimonial's author.
Testimonials: Social Profile URL - Paste the link to the author's Facebook or Twitter page, to let people check out the profile and make sure that the author is real.
Testimonials: Caption - Set any additional info you'd like to display, e.g. the author's company name or position.
Testimonials: Picture - Set the image URL to display alongside the testimonial. It could be the author photo or any other appropriate image.
Testimonials: Website URL - Set the URL of the author's website. It will be displayed under the caption. In case you've specified the company logo in the field below, the URL won't be displayed. Instead, the logo will become clickable and will take you to that website.
Testimonials: Company Logo - Paste the URL to the testimonial's author's company logo. In case you've specified the website URL in the field above, the logo will become clickable and will take you to that website.
Widget Title What our clients say Add a title to your testimonials slider widget.
